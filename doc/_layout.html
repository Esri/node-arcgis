Node ArcGIS

$ npm install arcgis

Client library for interfacing with ArcGIS Online and Enterprise. Works in server-side Node environments as well as the browser. Provides a set of convenience methods, and structures the underlying REST API concepts and behaviors to align closer with the expectations of an API in the node environment.
